# Parent Teacher Conference Form

A responsive and accessible **Parent Teacher Conference Form** created using **HTML5** and **CSS3**. This project allows parents or guardians to provide student details, choose a preferred contact method, and add discussion topics before scheduling a conference.

## Features

* Responsive centered form layout
* Student information collection
* Parent/Guardian information section
* Custom styled radio buttons
* Additional notes textarea
* Hover effects on submit button
* Modern UI using transparency and shadows

## Technologies Used

* HTML5
* CSS3
